# Daily Learning Log

このリポジトリは、日々の学習内容を記録するためのログです。
Web開発・インフラ・データベース設計を中心に、実務レベルの理解を目的としてアウトプットしています。

---

## 目的

* 学習内容を言語化し、理解を定着させる
* 知識を再利用できる形で蓄積する
* 設計や実装における思考プロセスを残す

本ログは、勤怠管理アプリ開発と並行して学習した内容を記録しています。

---

## 内容について

単なる知識のまとめではなく、以下を重視しています：

* なぜそうなるのか（背景・原理）
* どこで問題になるのか（不整合・設計ミス）
* 実装時にどう活きるか

理解途中の内容や仮説も含まれますが、思考の過程として記録しています。

---

## AIの活用について

学習内容の整理・要約にはAI（ChatGPT等）を補助的に使用しています。

ただし、

* 内容の理解・解釈は自分で実施
* AIの出力は必ず検証
* 自分の言葉で再構成して記録

という方針で運用しています。

AIはあくまで思考を補助するツールとして活用しています。
